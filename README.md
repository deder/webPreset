## **webStarter**

Ce projet a pour but de creer un starter pour mes projets :
Aujourd'hui, j'utilise une mixture contenant les substances suivante : 

 - Webpack : minification et la concaténation du code
 - Webpack-dev-server : avoir un serveur web liveReload
 - Babel : transpiler mon code de ES6-7 vers ES5

*Le fichier javascript starter de l'application en mode dev se nomme main.js*
*Le fichier javascript starter de l'application en mode prod se nomme bundle.js*

Afin d'éxécuter le server il faut :

 1 Télécharger et installer nodeJs et NPM
 2 Lancer la commande :  ## **npm install** **L'objectif est de télécharger toutes les dépendances du projet**
 3 Lancer la commande :  ## **npm run build** **L'objectif est de créer l'arborescence du projet et de builder le projet**
 4 Lancer la commande :  ## **npm start** **L'objectif est de lancer le serveur web**
